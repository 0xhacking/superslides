• Superslides is a full screen slider for jQuery.

    Download v0.2

    Full documentation is available at Github.

  • It's only requires a little bit of HTML.

    Inside each li you can add your own markup and design.

  • It's light on features, but easily configurable.

    Currently, Superslides has a few configurable options: auto play with a delay, slide transition speed, and slide easing (jQueryUI or the jQuery easing plugin required). 

    delay: 5000
play: false
slide_speed: 'normal'
slide_easing: 'linear'
nav_class: 'slides-navigation'

    There are even a few callbacks available.

    adjust_slides_size_callback
adjust_image_position_callback
animate_callback

  • You might even be interested in contributing.

    Fix a bug or add a feature by forking on Github.

    Superslides is a project by @nicinabox
